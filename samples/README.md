# Sample logs, metrics, and traces

This tree holds **reference JSON** generated by the same registries as **[Cloud to Elastic Load Generator](../README.md)**. Layout:

| Path                               | Contents                                                      |
| ---------------------------------- | ------------------------------------------------------------- |
| **`aws/{logs,metrics,traces}/`**   | One file per AWS generator (see app when **AWS** is selected) |
| **`gcp/{logs,metrics,traces}/`**   | One file per GCP generator                                    |
| **`azure/{logs,metrics,traces}/`** | One file per Azure generator                                  |

Per-cloud notes: [gcp/README.md](./gcp/README.md).

## Regenerate everything

From the repo root:

```bash
npm run samples
```

## Verify coverage

```bash
npm run samples:verify
```

Fails with a non-zero exit code if any expected sample is missing or unexpected files exist. Run after changing generators.
